# How to set up i3 and xfce  

### Install i3  
`sudo apt install i3`  

### Disable xfwm and xfdesktop  
go to start up and disable these two  
* xfwm  
* xfdesktop  

### set the config for i3  
`mkdir -p ~/.config/i3/config`  
use my config or default one.  

### put i3 in start up  
put i3 in start up at login  

### Reboot  

### Remove all xfce shortcuts  

### you are done!!!
